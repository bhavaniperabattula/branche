VS Code

Visual Studio Code is a source-code editor that can be used with a variety of programming languages, including C, C#, C++, Fortran, Go, Java, JavaScript, Node.js, Python, Rust, and Julia.Visual Studio Code employs the same editor component (codenamed "Monaco") used in Azure DevOps (formerly called "Visual Studio Online" and "Visual Studio Team Services").

Features

    1.Cross-Platform: Available on Windows, macOS, and Linux.
    2.Intelligent Code Completion: Features like IntelliSense provide smart code completion based on variable types, function definitions, and imported modules.
    3.Built-in Git Integration: Provides Git commands built-in, allowing you to manage your source control with ease.
    4.Extensions: A rich ecosystem of extensions to support various programming languages, frameworks, and tools.
    5.Debugging: Integrated debugging for various languages, including setting breakpoints, stepping through code, and inspecting variables.

HTML
  • HTML stands for Hyper Text Markup Language
  • It was introduced in 1991 by tim berners lee
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements

heading tags

Ranges from h1 tag to h6 tag, define headings of different levels, with h1 tag being the highest (most important) level and h6 tag the lowest.

Formatting tags
  1. Paragraph tag: used to define a paragraph of text
  2. Preformatted tag : It preserves whitespace and line breaks for displaying text exactly as it appears in the source code.
  3. Bold tag : Makes the text bold without adding any extra importance
  4. Italic tag : Makes the text italic without adding any extra emphasis
  5. Strong tag : Makes the text bold and indicates that the text is of strong importance
  6. Emphasize tag : Makes the text italic and indicates that the text is emphasized
  7. Underline tag : Underlines the text
  8. Subscript tag : Renders text as subscript (below the baseline)
  9. Superscript tag : Renders text as superscript (above the baseline)
  10. Deleted tag : Represents deleted text, commonly used to show text that has been removed
  11. Mark tag : It is used to highlight text. The highlighted text typically appears with a yellow background by default.
  12. Blockquote tag : It is used to represent text that is quoted from another website
  13. Abbreviation tag : It is used to define an abbreviation providing a full description when hovered over by the user.
  14. Quataiton tag : It is used to indicate quotations.
  15. Cite tag : It is used to denote the title of a work, such as a book, article, movie, or other creative work. The content within the cite tag is typically rendered in italicized text by default.
  16. Address tag : It is used to represent the address information
  17. Code tag : Represents a fragment of computer code with a monospace font
  18. Keyboard tag : It is used to indicate text that represents keyboard input
Blockline elements

Block-level elements in HTML are elements that typically start on a new line and occupy the full width available to them.

Inline elements

Inline elements in HTML are elements that do not start on a new line and only occupy as much width as necessary, allowing other elements to sit beside them horizontally.

Table tag
  • It is used to represent data in a tabular form
  • Tables are defined with table tag
  • Rows are represented with tr
  • Columns are represented with td
  • Column headings are represented with th

Table features
  1. Table caption : It is used to provide title for a table
  2. Cell padding : It is used to create spaace around content in a cell
  3. Cell spacing : It is used to create a space between cells
  4. Background color : It is used to apply background color to a table
  5. width and height : It is used to define height and width to an element

Structure of url
  1. Protocol: It is a set of rules and standards that define how data is transmitted and received over a network(e.g., http, https)
  2. Subdomain: A subdivision of the main domain (e.g., www in www.example.com)
  3. Second-Level Domain (SLD): The main domain name (e.g., example in example.com)
  4. Top-Level Domain (TLD): The highest level of the domain name system hierarchy (e.g., com in example.com)
  5. Port: The port number on the server (default ports are 80 for HTTP and 443 for HTTPS).
Image tag

The image tag in HTML is used to embed images into a webpage. It is an empty (self-closing) tag that requires the src attribute to specify the image source URL, and typically includes the alt attribute to provide alternative text if the image cannot be displayed
Anchor tag

The anchor tag is used to create hyperlinks, allowing users to navigate to other webpages, files, or locations within the same page. It typically includes the href attribute, which specifies the URL of the linked resource.
List tag

HTML lists allow web developers to group a set of related items in lists. There are 3 types of lists. They are
  1. Ordered List
  2. UnOrdered List
  3. description List
